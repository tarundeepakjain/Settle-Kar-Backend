# Settle Kar â€” Group Expense & Bill-Splitting Application (Backend)

Settle Kar is a Splitwise-like **group expense and bill-splitting backend system** designed to simplify shared expense management.  
This repository contains the **backend implementation**, built with a focus on clean architecture, correctness, and scalability.

## Tech Stack
- **Node.js**
- **Express.js**
- **MongoDB**

---

## Key Features

### Group Expense Splitting
- Create groups for shared expenses.
- Add expenses with multiple participants.
- Automatically calculate individual balances for each group member.
- Maintain accurate settlement records across transactions.

### Personal Expense Tracking
- Track individual (non-group) expenses separately.
- Maintain personal expense history alongside group expenses.

### Invite Code System
- Generate unique invite codes for groups.
- Allow users to join existing groups securely using invite codes.

### Core Functionalities
- User authentication and authorization.
- Change password functionality.
- Edit user and group details.
- View expense and transaction history.
- Secure and validated backend operations.

---

## Backend Design
- Implemented using **Object-Oriented Programming (OOPS)** principles.
- Modular architecture with clear separation of concerns.
- Robust validation to ensure data consistency across concurrent users.
- RESTful API design using Express.js.

---

## Database
- **MongoDB** is used for persistent storage.
- Stores users, groups, expenses, balances, and transaction history.
- Schema design optimized for group-based expense calculations.
